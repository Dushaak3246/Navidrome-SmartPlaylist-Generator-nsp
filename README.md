## 📄 What is Navidrome-SmartPlaylist-Generator-nsp?

Navidrome-SmartPlaylist-Generator-nsp is a simple tool that helps you make smart playlists for Navidrome. Navidrome is a music server where you can store and stream your music collection. Smart playlists in Navidrome automatically update based on rules you set, like “songs from the 90s” or “tracks I haven’t played recently.”

This tool guides you step-by-step to build these playlists. You don’t need to edit any code or remember special commands. It works through your command line (the black window where you type instructions) and creates `.nsp` files Navidrome understands.

- Think about what kind of music you want to hear or organize.  
- Use rules like artist name, date added, play count, or audio quality.  
- Combine rules with AND/OR conditions to fine-tune results.  
- Try sorting by song title or release year to get your list in order.
